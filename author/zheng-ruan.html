Motivation for the Blog

Mon 03 June 2013
By Zheng Ruan

In Me.

tags: motivationGSoC

Originally, I plan to use google blogger system for my GSoC Home Page, as Peter Cock suggested. However, I learnt from Yanbo Ye that github also support static blog system. I then spent a day to play with it and finally got the basic way to set it up. Pelican is different from CMS like wordpress, drupal and django as it focus on translate many text-to-html language, thus faciliate blogging. I like this idea as html is too cumbersome to work with. With some adjustment of them template (google fonts and some css element), the blog now looks fine and hope it will be a good start!

Zheng Ruan
